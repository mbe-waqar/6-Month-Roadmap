# MONTH 1: GenAI Foundations (December 2025)

**Theme:** Deep dive into transformer architectures and large language models  
**Weeks:** 1-4 (Dec 1 - Dec 28)  
**Total Hours:** 80 hours (20/week)

## Weekly Breakdown

### Week 1 (Dec 1-7): Advanced Transformers
Master the fundamental architecture powering modern AI. Implement transformers from scratch to understand attention mechanisms, positional encoding, and the encoder-decoder pattern. Build working implementations that can handle sequence-to-sequence tasks and visualize attention weights to gain intuition.

**Key Tasks:**
□ Implement multi-head attention in PyTorch (8h)
□ Build complete transformer architecture (6h)
□ Study attention mechanisms theory (4h)
□ Read "Attention Is All You Need" paper (2h)

**Deliverable:** Working transformer with documentation on GitHub

---

### Week 2 (Dec 8-14): Pre-training & Fine-tuning
Learn how modern language models are trained and adapted. Fine-tune BERT for classification and GPT-2 for generation, comparing different strategies. Implement LoRA for parameter-efficient fine-tuning to understand trade-offs between full and partial fine-tuning approaches.

**Key Tasks:**
□ Fine-tune BERT and GPT-2 on datasets (9h)
□ Implement LoRA from paper (6h)
□ Study pre-training strategies (3h)
□ Compare fine-tuning methods (2h)

**Deliverable:** Fine-tuned models with performance comparison

---

### Week 3 (Dec 15-21): Modern LLM Architectures
Explore state-of-the-art LLM designs including LLaMA, Mistral, and optimization techniques. Deploy a local LLM and implement quantization with GPTQ and AWQ. Learn inference optimization through KV caching and continuous batching to prepare for production deployments.

**Key Tasks:**
□ Deploy LLaMA 2 7B locally (8h)
□ Implement quantization methods (6h)
□ Study modern architectures (4h)
□ Optimize inference speed (2h)

**Deliverable:** Optimized local LLM with benchmarks

---

### Week 4 (Dec 22-28): Prompt Engineering & Agents
Master advanced prompting techniques and build autonomous AI agents. Implement chain-of-thought, ReAct, and tree-of-thoughts patterns. Create a multi-agent system using LangChain with custom tools and function calling to solve complex multi-step tasks.

**Key Tasks:**
□ Build multi-agent system (10h)
□ Create custom tools (6h)
□ Study agent architectures (2h)
□ Read ReAct paper (2h)

**Deliverable:** Multi-agent application with tool integration

---

## Month 1 Checklist
□ All 4 weeks completed on schedule  
□ 4 major deliverables on GitHub  
□ Transformer fundamentals mastered  
□ Ready for RAG and generative models
